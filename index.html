Projelerim

E-Ticaret Uygulaması

ASP.NET MVC ve Entity Framework kullanarak, MS SQL Server üzerinde kurgulanan ilişkisel veritabanı yapısıyla kullanıcı kimlik doğrulama, sepet yönetimi ve kapsamlı admin paneli modüllerinde tam CRUD operasyonlarını içeren uçtan uca bir sistem geliştirdim.

Havayolu Rezervasyon

ASP.NET ve SQL Server altyapısıyla, yüksek tutarlılık gerektiren uçuş planlama ve biletleme süreçlerini yöneten kurumsal bir web çözümü geliştirdim; rol tabanlı yetkilendirme ve dinamik kontenjan yönetimi modülleriyle veri akışını optimize ettim.

Stok & Depo Yönetimi

Java ve Spring Boot ile RESTful API mimarisinde geliştirilmiş backend sistemi; JWT tabanlı güvenlik, stok giriş-çıkış takibi ve minimum stok uyarı mekanizmaları içeren ölçeklenebilir ve tutarlı bir kurumsal envanter yönetim mimarisi oluşturdum.

Kişisel Web Sitesi

HTML5, CSS3 ve Vanilla JavaScript kullanarak modern tasarım prensipleriyle geliştirdiğim; responsive yapıda, kullanıcı deneyimi odaklı, neon estetiği ve dinamik animasyonlar içeren profesyonel portfolyo web sitesi projemdir.
